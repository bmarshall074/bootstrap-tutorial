Overview

Bootstrap is a CSS and Javascript library coded by developers of Twitter and made freely available to the public at http://twitter.github.com/bootstrap. This library allows you to get some quick styling and interesting interactions in your site or web app without much design work needed on your part.

Basic Usage

In general, to use all that Bootstrap has to offer, you'll have to do the following:

  1. Download Bootstrap and extract it to your project
  2. In your HTML file's <head> element...
    • Link to the Bootstrap CSS file that is included in your extract files: 
      <link rel="stylesheet" type="text/css" href="bootstrap/css/bootstrap.css" />
    • Reference the jQuery (must appear first) and Bootstrap Javascript files: 
      <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.2/jquery.min.js"></script>
<script type="text/javascript" src="bootstrap/js/bootstrap.js"></script>
  3. Include the required markup and any attributes required to produce your desired styling or effect.

Dropdown Menus Dropdowns on Bootstrap

You can produce dropdown menus with the simple addition of a the attribute data-toggle="dropdown" to an anchor that is a sibling to a <ul> as follows:

Markup

<div class="dropdown">
	<a class="dropdown-toggle" data-toggle="dropdown" href="#">Dropdown</a>
	<ul class="dropdown-menu">
		<li><a href="#">Item 1</a></li>
		<li><a href="#">Item 2</a></li>
		<li><a href="#">Item 3</a></li>
	</ul>
</div>

CarouselsCarousels on Bootstrap

The use of carousels allows you to drop a slideshow in your site that visitors can page through. This is accomplished with the use of a wrapper div with class="carousel", and a child div with class="carousel-inner", itself containing child div's with class="item", each representing a "slide" of the carousel.

Markup

<div id="myCarousel" class="carousel slide">
	<!-- Carousel items -->
	<div class="carousel-inner">
		<div class="active item">
			<img src="http://placehold.it/940x400/57aade/ffffff&text=Image+1">
			<div class="carousel-caption">
				<h4>Sample Title 1</h4>
				<p>Sample description for IMAGE 1. This is just a test.</p>
			</div>
		</div>
		<div class="item">
			<img src="http://placehold.it/940x400/e8117f/ffffff&text=Image+2">
			<div class="carousel-caption">
				<h4>Sample Title 2</h4>
				<p>Sample description for IMAGE 2. This is just a test.</p>
			</div>
		</div>
		<div class="item">
			<img src="http://placehold.it/940x400/FF6B2B/ffffff&text=Image+3">
			<div class="carousel-caption">
				<h4>Sample Title 3</h4>
				<p>Sample description for IMAGE 3. This is just a test.</p>
			</div>
		</div>
	</div>
	<!-- Carousel nav -->
	<a class="carousel-control left" href="#myCarousel" data-slide="prev">&lsaquo;</a>
	<a class="carousel-control right" href="#myCarousel" data-slide="next">&rsaquo;</a>
</div>
